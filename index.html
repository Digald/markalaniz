Projects

  • Mark's Body-Works

    A full-stack application that is a collection of my favorite weightlifting programs all in one place. It's best suited for intermediate lifters. This Meteor app handles all of the user's data while React in the front-end renders your program specs smooth and quickly.

  • Skiller-bot

    A private bot for my friend's personal discord server. While the bot is online, Skiller-bot listens for specified commands and returns various data or information, depending on what the user asks for. Commands are being actively developed based on the needs of our group, and the file structure of the bot is open to change.

  • RuneScrape

    A MongoDB web-scraper app to bring you the latest Old School Runescape news all in one place. Save articles you'd like to keep and have the ability to comment on them.

  • JoJo's React Adventures

    A single page memory game application built with React. This game shows off the basic functionality of React by utilizing props, state, and event handlers to take advantage of the virtual DOM.

  • BitBay

    Bitcoin had a huge peak in late 2017 which sparked the idea for a webpage. BitBay has the ability to track bitcoin statistics and information. Additionally, users may also search for their favorite products on ebay and compare the cost in USD to Bitcoin.
